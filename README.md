# рџљЂ Installation

## 1 Clone Repository

```bash
git clone https://github.com/kiranlondhe432005/AI-Travel-Planner.git
```

---

## 2 Navigate

```bash
cd AI-Travel-Planner
```

---

## 3 Install Frontend

```bash
cd frontend
npm install
```

---

## 4 Install Backend

```bash
cd ../backend
npm install
```

---

## 5 Environment Variables

Create a `.env` file inside the backend folder.

```
PORT=5000

MONGODB_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET

GROQ_API_KEY=YOUR_GROQ_API_KEY
```

---

## 6 Start Backend

```bash
npm run dev
```

---

## 7 Start Frontend

```bash
cd ../frontend

npm run dev
```
